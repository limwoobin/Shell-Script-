# Shell-Script


```shell script
/ root(/)
  /bin - 
  /sbin - 시스템 권한을 갖는 루트 사용자가 사용하는 명렁어들이 들어있는 디렉터리
  /usr - window 의 ProgramFiles 와 같은 많은 응용프그램들이 설치되는 개념
    /bin - 리눅스 운영체제에 설치된 많은 응용프로그램이 위치 , 명령어로 존재 , 응용프로그램의 바이너리 파일들이 위치
    /lib - 응용프로그램이 설치될 당시 실행가능 프로그램외에 사용하는 라이브러리가 위치
    /local - 프로그램 설치는 /usr 외에도 /usr/local 디렉터리에도 설치될 수 있음
  /dev - 장치 파일이라 불리는 많은 파일이 존재 (리눅스의 특수한 파일이라 불리는 형태) , 장치파일들을 통해 리눅스에 설치되있는 많은 하드웨어들이 작동
  /etc - 환경 파일이라는 것들이 모여있음 , window 의 registry 와 같은 개념
  /var - 시스템에 관련된 많은 정보들을 저장
    /log - 시스템 내부의 동작상황을 모니터링할수 있는 로그파일을 저장
```
